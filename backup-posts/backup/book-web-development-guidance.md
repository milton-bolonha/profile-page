# Web Development
## Essential Guidance for Beginners and Web Agencies



O guia definitivo de desenvolvimento web moderno. Tudo que você precisa saber para criar sites profissionais em 2025, seja você iniciante ou agência querendo se atualizar.

## Por Que Este Livro?

O mercado de desenvolvimento web mudou radicalmente. HTML, CSS e jQuery não são mais suficientes. Mas a maioria dos cursos ou ensina demais (frameworks complexos) ou de menos (só o básico).

Este livro preenche essa lacuna: **ensina o necessário para ser produtivo hoje**, sem perder tempo com tecnologias obsoletas.

## Para Quem É

### Iniciantes
- Quer aprender desenvolvimento web do zero
- Tem noções básicas mas quer se profissionalizar
- Está mudando de carreira para tech
- Quer trabalhar como freelancer

### Agências
- Usa WordPress mas quer modernizar
- Quer oferecer soluções mais performáticas
- Precisa treinar equipe em tecnologias modernas
- Quer aumentar ticket médio com projetos melhores

## O Que Você Vai Aprender

### Fundamentos (Sem Enrolação)
- HTML5 semântico (o que realmente importa)
- CSS moderno (Flexbox, Grid, variáveis)
- JavaScript essencial (ES6+, async/await, fetch)
- Git e GitHub (workflow profissional)
- Terminal e linha de comando

### Ferramentas Modernas
- **VS Code**: Setup e extensões essenciais
- **Node.js e npm**: Gerenciamento de pacotes
- **Vite**: Build tool rápido
- **Tailwind CSS**: CSS utilitário
- **TypeScript**: Quando e por que usar

### Frameworks (O Que Importa)
- **React**: Fundamentos e hooks
- **Next.js**: SSR, SSG, App Router
- **Componentes**: Reutilização e organização
- **Estado**: useState, useContext, Zustand
- **Formulários**: React Hook Form

### Performance e SEO
- Core Web Vitals explicados
- Otimização de imagens (WebP, lazy loading)
- Code splitting e lazy loading
- SEO técnico para desenvolvedores
- Lighthouse e ferramentas de auditoria

### Deploy e Infraestrutura
- **Vercel**: Deploy de Next.js
- **Netlify**: Sites estáticos
- **GitHub Pages**: Projetos simples
- **Domínios e DNS**: O básico que você precisa
- **SSL/HTTPS**: Configuração

### Trabalhando com Clientes
- Como precificar projetos
- Briefing e levantamento de requisitos
- Contratos e propostas
- Gestão de expectativas
- Manutenção e suporte

## Projetos Práticos

Cada seção tem projetos reais que você vai construir:

1. **Landing Page**: HTML, CSS, JavaScript
2. **Portfólio Pessoal**: React + Tailwind
3. **Blog**: Next.js + MDX
4. **E-commerce**: Next.js + Stripe
