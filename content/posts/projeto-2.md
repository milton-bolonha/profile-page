# Projeto 2 - Dashboard Analytics

Este é um projeto privado desenvolvido para cliente corporativo.

## Visão Geral

Dashboard completo de analytics com visualização de dados em tempo real, desenvolvido com tecnologias modernas.

## Tecnologias

- Next.js
- React
- Chart.js
- WebSockets para dados em tempo real
- PostgreSQL

## Funcionalidades Principais

- Visualização de métricas em tempo real
- Relatórios customizáveis
- Exportação de dados
- Sistema de alertas
- Multi-tenancy
