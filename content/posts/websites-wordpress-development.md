# WordPress Development

Desenvolvo websites profissionais em WordPress desde 2009. Já foram centenas de projetos, curadoria responsável, soluções customizadas, otimizadas para alta performance e escaláveis.

## Experiência

Trabalho com WordPress desde os primeiros anos da plataforma. Mais de 20 anos de experiência em desenvolvimento web me dão uma visão única: sei quando usar WordPress e quando outras soluções são melhores.

## Serviços Oferecidos

### Sites Customizados
- Desenvolvimento de temas do zero
- Child themes otimizados
- Design responsivo
- Gutenberg blocks customizados
- ACF (Advanced Custom Fields)
- Elementor/WPBakery quando necessário

### E-commerce (WooCommerce)
- Setup completo de loja virtual
- Customização de checkout
- Integração com gateways de pagamento
- Cálculo de frete
- Gestão de estoque
- Relatórios e analytics

### Plugins Customizados
- Desenvolvimento de funcionalidades específicas
- Integração com APIs externas
- Automações
- Dashboards administrativos
- Importação/exportação de dados

### Otimização e Performance
- Lighthouse score 90+
- Lazy loading de imagens
- Minificação de CSS/JS
- Database optimization
- Cache configuration (WP Rocket, W3 Total Cache)
- CDN setup (Cloudflare, Cloudinary)

### SEO
- Yoast SEO ou Rank Math configurado
- Schema markup
- Sitemap XML
- Otimização de URLs
- Meta tags automáticas
- Breadcrumbs

### Segurança
- Hardening do WordPress
- Firewall (Wordfence, Sucuri)
- Backup automático
- 2FA para admin
- Proteção contra brute force
- Atualizações regulares
